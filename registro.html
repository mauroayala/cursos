Curso de Diseño Web

En este curso aprenderás a crear tu propio sitio web, trabajando con HTML y CSS
Para este curso no es necesario tener conocimientos previos. El curso inicia en Marzo de 2023.

Modalidad

- Online en vivo: vas a tener 2 clases en vivo por semana.

Duración

10 Clases, de una hora . Luego se podrán dejar todas las consultas necesarias.

Responsive

Aprenderás cómo armar sitios web para pc y celulares.

Contenido del Curso

Introducción

Presentación / Navegadores / Clientes y Servidor / Herramientas / Sitios

HTML

Primeros pasos con HTML / Estructuras y etiquetas

CSS

Maquetación / Grid / Posicionamiento / Diseño de tablas / Selectores

Frameworks

Frameworks CSS + Bootstrap

Publicar sitio web

Hosting / Dominio / Google / FTP

Duración

10 Clases, cada clase durará 1 hora a las 21:30

Tarea Semanal

Posee una carga de actividades a realizar por semana.

Clases en VIVO

Luego de las clases se podrán hacer todas las preguntas que surjan.

Medios de Pago

Tarjeta de crédito, débito, transferencia bancaria o efectivo.

Dificultad

INICIAL.

Requisitos mínimos

Computadora 2 gb de RAM y una cámara web.

Proyecto Final

Desarrollarás una landing page de un tema a tu elección. La web será diseñada enteramente por tí, utilizando todo lo aprendido en clase.

Finalizarás el curso con un producto terminado y subido a internet.
